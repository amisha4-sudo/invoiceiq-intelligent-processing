# invoiceiq-intelligent-processing
InvoiceIQ is an intelligent document processing system that extracts key information from invoices using OCR technology.

## Features
- Invoice text extraction  
- OCR-based processing  
- Simple UI  

## Tech Stack
- Python  
- Tesseract OCR  
- Streamlit  

## Run
streamlit run app.py

## How It Works
- Upload invoice image  
- Extract text using OCR  
- Display extracted content  

## Future Improvements
- Field extraction (amount, date)  
- Fraud detection
